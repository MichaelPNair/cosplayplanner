# CosplayPlanner

This is a website to plan cosplay projects. You can add, edit and delete multiple elements of a cosplay plan to quickly gauge your progress or adjust if the plans are too difficult or too expensive.

## Technologies Used
- HTML
- CSS
- Javascript
- Postgres
